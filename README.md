HQLHelper.java文件是分页的sql语句的帮助类，用于得到分页所需的sql语句，通过链式调用可以添加多个where条件以及order by条件，通过这些条件来得到用于分页查询的sql语句，并且可以使用prepareBean()方法利用以上sql语句得到分页要用的pageBean对象并放在值栈的栈顶。
